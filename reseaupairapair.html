Fonctionnement du DNS

  • Résolution des noms : Lorsqu'un utilisateur entre un nom de domaine dans le navigateur, le DNS se charge de trouver l'adresse IP associée à ce domaine.
  • Serveurs DNS : Il existe plusieurs types de serveurs DNS qui travaillent ensemble pour compléter la résolution des noms, comme les serveurs racine, les serveurs TLD (Top-Level Domain) et les serveurs autoritatifs.
  • Cache DNS : Pour améliorer l'efficacité, les réponses DNS sont temporairement stockées dans des caches, permettant une résolution plus rapide des noms déjà consultés.

Composants du DNS

Le DNS se compose de plusieurs éléments clés :

  • Zone : Une portion de l'espace de noms de domaine qui est administrée par une entité spécifique.
  • Enregistrement DNS : Une entrée dans la base de données DNS qui fournit des informations sur un domaine ou une adresse IP.
  • Serveur de noms : Un serveur qui stocke les enregistrements DNS et répond aux demandes de résolution des noms.

Types d'enregistrements DNS

  • Enregistrement A : Mappe les noms de domaine aux adresses IPv4.
  • Enregistrement AAAA : Mappe les noms de domaine aux adresses IPv6.
  • Enregistrement CNAME : Alias d'un nom de domaine à un autre.
  • Enregistrement MX : Désigne les serveurs de messagerie pour le domaine.
  • Enregistrement TXT : Contient des informations textuelles utilisables pour diverses fonctions, comme la vérification SPF.
