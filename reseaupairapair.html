Réseau pair à pair

Qu'est-ce que le réseau pair à pair?

    Le réseau pair-à-pair, ou peer-to-peer (P2P) en anglais, est un modèle de réseau informatique qui s’appuie sur internet, où chaque ordinateur sert à la fois d’émetteur (il distribue les données numériques) et de récepteur (il reçoit les données numériques). On ne distingue pas les ordinateurs au sein d’un même réseau, c’est pourquoi on dit que ce modèle fonctionne d’égal à égal. Plusieurs ordinateurs peuvent donc communiquer entre eux via un réseau et échanger diverses informations.

Les utilisations possibles des réseaux pair-à-pair

Les réseaux pair-à-pair sont utilisés dans le partage d’informations diverses entre les membres du réseau : il peut s’agir, par exemple, d’échanges de fichiers ou alors de transactions d’argent.

  • Partage de fichiers Une portion de l'espace de noms de domaine qui est administrée par une entité spécifique.
  • Enregistrement DNS : Une entrée dans la base de données DNS qui fournit des informations sur un domaine ou une adresse IP.
  • Serveur de noms : Un serveur qui stocke les enregistrements DNS et répond aux demandes de résolution des noms.

Types d'enregistrements DNS

  • Enregistrement A : Mappe les noms de domaine aux adresses IPv4.
  • Enregistrement AAAA : Mappe les noms de domaine aux adresses IPv6.
  • Enregistrement CNAME : Alias d'un nom de domaine à un autre.
  • Enregistrement MX : Désigne les serveurs de messagerie pour le domaine.
  • Enregistrement TXT : Contient des informations textuelles utilisables pour diverses fonctions, comme la vérification SPF.

Voici un exemple d'enregistrement A :

example.com. 3600 IN A 192.0.2.1
